BlackboxEditor — Windows sideload install
==========================================

1. Extract this zip to a folder.
2. Double-click install-msix-sideload.cmd (or run install-msix-sideload.ps1).
3. Approve the UAC prompt. Admin rights are required once to trust the dev certificate.

The script installs msix-signing.cer into Trusted People, then installs the MSIX package.
